## TROUBLESHOOTING QUICK FIXES

### Blog Post Not Ranking
```
Check:
□ Indexed? (site:domain.com/blog/[slug] in Google)
□ Mobile friendly? (Google Mobile-Friendly Test)
□ Fast enough? (Lighthouse 80+)
□ Schema valid? (Schema.org validator)
□ Has enough internal links? (At least 3-5)
□ Primary keyword in H1? (Yes)
□ Unique content? (Not duplicate)
□ Old posts have similar keyword? (May cannibalize)

Fix:
1. Fix indexing (submit to GSC)
2. Improve speed (compress images, minify code)
3. Add schema markup
4. Add 5 more internal links
5. Improve content depth (+500 words)
6. Wait 2-4 weeks (new content takes time)
```

### Low CTR from Search Results
```
Check:
□ Title compelling? (Does it stand out vs competitors?)
□ Description has benefit? (User sees value?)
□ Preview looks good? (Test in Google Preview tool)
□ Keywords match query? (Is it relevant?)
□ Rich snippets showing? (Review score, FAQ, etc)

Fix:
1. Rewrite title with benefit + keyword
2. Add number or urgency to title ("2026 Guide", "Latest", etc)
3. Rewrite description with CTA
4. Add rich snippets (FAQ, review, recipe schema)
5. Test new version (wait 1-2 weeks for Google to update)
```

### Competitor Outranking for Easy Keyword
```
Analyze:
□ Competitor content length (yours longer by 500+ words?)
□ Backlinks (how many do they have?)
□ Domain age (are they older?)
□ Page speed (yours faster?)
□ Internal links (yours have more?)
□ Featured snippet (do they have one?)

Strategy:
1. Add 1000 words to your content (make it 2x longer)
2. Add video to your content (they may not have)
3. Add data/statistics (more authoritative)
4. Build 5-10 backlinks (increase authority)
5. Optimize for featured snippet
6. Improve meta tags + description CTR
7. Get internal links from homepage
```
